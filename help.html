How to Upload Projects

Step-by-step guide for admin panel

Step 1: Login to Admin Panel

  1. Go to Admin Login Page
  2. Enter username: admin
  3. Enter password: shagun123
  4. Click "Login" button

Step 2: Upload a Project

  1. Click on "Upload Project" in the left sidebar
  2. Fill in the Project Title (required)
  3. Select a Category from dropdown (required):
    • VFX
    • Video Editing
    • AI Films
    • Motion Graphics
  4. Select an Image file for thumbnail (required)
    Important: Image must be less than 2MB. Supported formats: JPG, PNG, WebP
  5. (Optional) Select a Video file
    Important: Video must be less than 10MB. Supported formats: MP4, WebM
  6. Add a Description about your project (optional but recommended)
  7. Click "Upload Project" button
After upload, you'll see a green success message. The project will appear on the Projects page and Homepage automatically!

Step 3: View Your Projects

  1. Click "Manage Projects" in the sidebar to see all uploaded projects
  2. You can delete projects from here using the trash icon
  3. Go to Projects Page to see how visitors will see your work

Step 4: Check Messages

  1. Click "Messages" in the sidebar
  2. See all contact form submissions from visitors
  3. Reply to emails directly from the email addresses shown

Tips for Best Results

  • Use high-quality but compressed images for faster loading
  • Recommended thumbnail size: 800x600 pixels
  • Keep videos short (under 10MB) for better performance
  • Use descriptive titles so visitors understand your work
  • Categorize correctly so projects appear in right filters
