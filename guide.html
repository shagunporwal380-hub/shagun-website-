How to Add Photo & Logo

Step-by-step guide for About page

Part 1: Add Your Logo

1

Create/Prepare Your Logo

Logo banayein Canva, Photoshop, ya kisi bhi tool se. Size: 500x500 pixels recommended. Format: PNG (transparent background ke liye) ya JPG.

2

Save Logo in images Folder

Apne logo ko images/ folder mein rakhein. Example filename: my-logo.png

Folder structure:
shagun-portfolio/
├── images/
│ ├── my-logo.png <-- Aapka logo yahan
│ └── shagun-photo.jpg
├── about.html
└── ...
3

Edit about.html

about.html file kholiye aur neeche wala code dhoondiye (around line 95):

<!-- LOGO YAHAN ADD KARO -->
<div class="about-logo">
    <i class="fas fa-film"></i>
</div>

Isko replace karein:

<div class="about-logo">
    <img src="images/my-logo.png" alt="Shagun Porwal Logo">
</div>
Tip: Agar aapke paas logo nahi hai, toh default film icon rehne dein. Ya text-based logo use karein.

Part 2: Add Your Photo

1

Choose a Professional Photo

Ek achhi professional photo select karein. Recommended: Portrait/vertical photo (3:4 ratio). Size: 800x1000 pixels max. Format: JPG.

2

Save Photo in images Folder

Photo ko images/ folder mein rakhein. Example: shagun-photo.jpg

3

Edit about.html

about.html mein neeche wala code dhoondiye (around line 115):

<!-- PHOTO YAHAN ADD KARO -->
<div class="about-photo-frame">
    <div class="about-photo-placeholder" id="photoPlaceholder">
        ...placeholder...
    </div>
    <!-- Uncomment neeche wali line: -->
    <!-- <img src="./images/shagun-photo.jpg" class="about-photo" alt="Shagun Porwal"> -->
</div>

Comment hata ke aise karein:

<div class="about-photo-frame">
    <img src="images/shagun-photo.jpg" class="about-photo" alt="Shagun Porwal">
</div>
Important: Photo ka naam aur src="..." mein naam exactly same hona chahiye. Case sensitive hai!

Quick Code Reference

About page mein 2 jagah edit karna hai:

1. Logo Section (Line ~95)

<!-- Option A: Image Logo -->
<div class="about-logo">
    <img src="images/YOUR-LOGO.png" alt="Logo">
</div>

<!-- Option B: Icon Logo (Default) -->
<div class="about-logo">
    <i class="fas fa-film"></i>
</div>

2. Photo Section (Line ~115)

<!-- With Photo -->
<div class="about-photo-frame">
    <img src="images/YOUR-PHOTO.jpg" class="about-photo" alt="Shagun Porwal">
</div>

<!-- Without Photo (Placeholder) -->
<div class="about-photo-frame">
    <div class="about-photo-placeholder">
        <i class="fas fa-user"></i>
        <p>Add Your Photo</p>
    </div>
</div>
